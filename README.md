## Project Description
Golden Castle is a turn-based strategy game developed in Java using Object-Oriented Programming principles.  
The game is played on a grid-based map where the player competes against an AI opponent by managing resources, building structures, training units, and engaging in combat.

This project was developed as part of a university course in Object-Oriented Programming.

---

## Game Features
- Turn-based gameplay (player vs AI)
- Grid-based map with different terrain types
- Resource management (Gold, Wood, Stone, Food)
- Buildings (Castle, Barracks, Farm, Mine)
- Units (Soldier, Archer, Cavalry)
- Unit movement with range rules
- Combat system (unit vs unit, unit vs building)
- Resource collection system
- Artificial Intelligence opponent
- Save and Load game system
- Graphical User Interface using Java Swing
